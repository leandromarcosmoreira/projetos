Apagar Todos os Dados

Esta ação irá TRUNCAR todas as tabelas de tickets, clientes, ações e históricos. Os dados serão perdidos permanentemente, a menos que você tenha um backup.

Tabelas de configuração e logs de sistema serão preservados.

Reprocessar Dados (Offline)

Re-executa a importação de dados utilizando os logs de API salvos (JSONs brutos). Útil para corrigir falhas de processamento sem consumir cota de API externa.

O sistema irá ler a tabela sys_offline_buffer e reinserir os registros.
